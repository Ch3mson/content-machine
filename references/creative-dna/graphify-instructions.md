# Graphify Instructions: Creative DNA Graph

Use these instructions when building a graph for content ideation in this repo.

## Purpose

Graphify is not being used here just to visualize the repo. It is being used as a
creative memory system.

The graph should help retrieve and connect reusable viral principles across
niches, then attach those principles to our own account constraints and post
workflows.

## Extraction Priority

Prioritize these graph nodes over file/folder nodes:

1. Viral Reference Post
2. Reusable Principle
3. Psychological Mechanism
4. Emotional Lever
5. Hook Archetype
6. Proof Pattern
7. Transformation Arc
8. Slide Sequence
9. Visual Pattern
10. Curiosity Gap
11. VOC Pattern
12. Adaptation Opportunity
13. Account Constraint
14. Target Account
15. Possible Post Concept

Files and headings are allowed as provenance, but they should not dominate the
graph.

## Edge Vocabulary

Use these relationship names whenever possible:

```text
reference_post → expresses → reusable_principle
reference_post → uses → hook_archetype
reference_post → creates → curiosity_gap
reference_post → relies_on → emotional_lever
reference_post → demonstrates → visual_pattern
reference_post → follows → slide_sequence
reference_post → proves_with → proof_pattern

reusable_principle → uses → psychological_mechanism
reusable_principle → maps_to → hook_archetype
reusable_principle → appears_in → niche
reusable_principle → transferable_to → target_account
reusable_principle → constrained_by → account_constraint
reusable_principle → suggests → possible_post_concept

reference_post_a → shares_mechanism_with → reference_post_b
principle_a → semantically_similar_to → principle_b
possible_post_concept → must_pass → quality_gate
possible_post_concept → shaped_by → account_writing_rules
possible_post_concept → shaped_by → account_design_rules
```

## Cross-Niche Examples To Look For

Extract and connect examples like:

- study productivity hook → transferable to soccer confidence post
- fitness transformation proof pattern → transferable to athlete story post
- men’s health red-flag format → transferable to athlete performance mistake post
- business status/identity angle → transferable to sports identity content
- discipline proof stack → transferable to athlete-max confidence or effort posts

## Bad Graph Output

Avoid stopping at:

```text
this file links to this file
this post belongs to this folder
this account is about soccer
```

That is provenance, not creative DNA.

## Good Graph Output

Prefer connections like:

```text
this study post and this soccer post use the same psychological mechanism
this fitness hook can be adapted to athlete stories
this men’s health red-flag format can become a performance mistake post
this viral proof stack maps well to athlete-max because it fits the account's identity pressure and tactical self-improvement voice
```

## Useful Questions The Graph Should Answer

- What viral patterns from other niches fit this account?
- What principle made this post work?
- Where else have we seen this mechanism?
- Which of our accounts could use this structure?
- What account rules would change how we adapt it?
- What emotional lever should this broad idea use?
- Which reference post should inspire this concept without being copied?
