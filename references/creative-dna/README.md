# Creative DNA Memory

This folder is the source of truth for turning viral references into reusable
creative principles. It exists so Graphify can connect **why a post worked** to
the accounts and workflows that can reuse the mechanism.

The goal is not to visualize files. The goal is to build a creative memory
system that helps us generate better account-specific post concepts faster.

## Core Job

Extract the transferable mechanism behind a viral/reference post:

```text
viral reference post
→ reusable creative principle
→ psychological mechanism
→ hook archetype
→ emotional lever
→ slide structure
→ visual pattern
→ transferable account fit
→ possible post concept for our accounts
```

## What Counts As Creative DNA

Create first-class concepts for:

- Reusable principle
- Psychological mechanism
- Emotional lever
- Hook archetype
- Proof pattern
- Transformation arc
- Slide sequence
- Visual pattern
- Curiosity gap
- VOC / audience-language pattern
- Transfer opportunity
- Account constraint
- Possible post concept

Do **not** stop at surface tags like `soccer`, `fitness`, or `study`. Those are
niches. The valuable node is the deeper mechanism that can travel across niches.

## Extraction Workflow

1. Start from a viral/reference post, hook card, transcript, slideshow, or social
   account post index row.
2. Fill `templates/viral-post-creative-dna.md`.
3. Save the filled extraction to `extractions/{source-slug}.md`.
4. When adapting it into a real post, connect the extraction to:
   - target account rules (`writing.md`, `design.md`, `image.md`, `sources.md`)
   - relevant workflow/skill (`post-concept-flow`, `angle-variants`, Workflow B)
   - hook memory (`references/hook-ideas/`)
5. Re-run the Creative DNA Graph so the principle becomes retrievable.

Use `creative-dna-workflow-router.md` when choosing which workflow or skill should
turn an extracted principle into a real post concept.

## Relationship To Hook Ideas

`references/hook-ideas/` captures useful hooks and retention mechanics.

`references/creative-dna/` goes one level deeper. It asks:

- What principle made this post work?
- What psychological mechanism is being used?
- Where else have we seen this mechanism?
- Which of our accounts could use it?
- What account rules would change the adaptation?

Hook cards can be inputs to Creative DNA extractions. Creative DNA extractions
should not copy source hooks verbatim; they should abstract the reusable move.

## Success Criteria

This system is useful when it can answer:

- What viral patterns from other niches fit this account?
- What principle made this post work?
- Where else have we seen this mechanism?
- Which of our accounts could use this structure?
- What account rules would change how we adapt it?
