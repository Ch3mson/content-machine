# Creative DNA Account Profiles

This folder connects reusable Creative DNA mechanisms to our owned accounts.

Use these profiles when the question is:

> “What content ideas should we make for this account?”

Each profile summarizes:

- audience tension
- approved voice/design/image constraints
- existing proven account patterns
- best-fit Creative DNA mechanism families
- weak or risky mechanism fits
- account-specific concept seeds
- canonical Graphify node labels

## Authority Rule

Account files remain higher authority than these profiles.

If a profile conflicts with `accounts/{account}/writing.md`, `design.md`,
`image.md`, `sources.md`, or `presets.md`, follow the account file.

Creative DNA profiles are retrieval aids, not final copy rules.
