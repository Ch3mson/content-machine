# Design Image Inputs

Add Pinterest images and visual references here.

Accepted inputs:

- Pinterest screenshots.
- Saved image files.
- Typography references.
- Overlay references.
- Layout references.
- Texture, crop, color, or image treatment references.

Extraction should identify:

- Typography hierarchy.
- Font feel.
- Text placement.
- Line breaks.
- Margins and safe zones.
- Image crop behavior.
- Overlay behavior.
- Color and contrast.
- Texture or grain.
- Product image fit, if relevant.
- Which visual rules vary by slide role.
