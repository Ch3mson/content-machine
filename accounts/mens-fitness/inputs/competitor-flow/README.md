# Competitor Flow Inputs

Add competitor slideshow references here.

Accepted inputs:

- Slide screenshots in order.
- A folder per competitor post.
- A text transcript of each slide.
- A link file with URLs and notes.
- Your notes on why the flow worked.

Recommended naming:

- `brand-or-account_post-topic_01.png`
- `brand-or-account_post-topic_02.png`
- `brand-or-account_post-topic_notes.md`

Extraction should identify:

- Hook type.
- Slide count.
- Slide roles.
- Pacing.
- Tension curve.
- What repeats.
- What changes from slide to slide.
- Any native CTA or product mention behavior.
- Whether the flow can support a soft app mention.
