# Quality Gate Attempts

Each new athlete-max quality-gate sample gets its own folder here.

Folder format:

```text
attempt-###-YYYY-MM-DD-{short-topic-slug}/
```

Required files per attempt:

- `attempt-summary.md`
- `sample-brief.md`
- `sample-copy.md`
- `render_sample.py`
- `sample-slide.png`
- `iteration-notes.md`

Update `../index.md` whenever an attempt is created, approved, rejected, or
superseded.
