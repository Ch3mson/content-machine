# Banned Patterns

## Banned Moves

| Pattern | Why It Fails | Replacement |
| --- | --- | --- |
| Generic motivation quotes | The user explicitly banned this. Makes the account sound like every other sports account. Feels AI-generated. | Specific, uncomfortable truths. "The guy taking your spot is doing extra reps right now." |
| "Remember that..." openings | Throat-clearing. Wastes the first slide on nothing. Adds zero value. | Start with the specific behavior or frustration directly. |
| Abstract value propositions | "This will change your game" — too vague. No accountability. | Name the specific change: "Do this in your next training session and watch what happens to your first touch." |
| Two ideas per slide | Splits attention. Reduces save impulse. Makes the design fight the copy. | One idea per slide. If two ideas, two slides. |
| "It's important to..." framing | Passive. Sound like a manual or a coaching certification. Not a player talking to a player. | Direct imperative or behavioral statement. "Cover your near post first. Everything else comes after." |
| Universal truth claims | "Every pro does this" — unverifiable and sounds fake. Not all pros do anything. | Observation framing: "The best defenders I've played with all do this one thing differently." |
| Question hooks that don't deliver | "Want to be a better player?" — generic, no tension, no specific promise. | Identity or tension hook: "How to be the player coaches trust in the 85th minute." |
| Ending with "Save this for later" or "Follow for more" | Lazy close. Interrupts the content. The save impulse should feel organic. | End on the most impactful line. Let the content earn the save. |
| "Let's be honest" / "Let's be real" | Unnecessary framing. Insults the reader's intelligence. | Just state the honest truth. The reader will recognize it. |
| Over-polished transitions | "Not only that, but..." / "Furthermore..." — AI sentence starters. Breaks the native feel. | Periods. Then the next sentence. No transitions needed. |
| Mini-essay arcs | The reference user accounts are list-first. A slow hook → lesson → test → moral flow feels less native to TikTok/Instagram user carousels. | Convert to hook + 3-5 listed mistakes, rules, habits, contrasts, or fundamentals. |

## Banned Phrases

- "Remember that..."
- "It's important to..."
- "Let's be honest/real"
- "At the end of the day"
- "Hard work beats talent"
- "There's no substitute for hard work"
- "The beautiful game" (unless ironic)
- "Game-changer" (unless describing a specific thing)
- "Next level" as a generic intensifier
- Any form of "unlock your potential"
- "Mindset is everything"
- "Trust the process"
- "The grind never stops"
- "Wake up and grind"
- "Be the best version of yourself"

## Weak Structures

- Question → generic advice → call to action (overused AI pattern)
- "X things every player should know" (overused, sounds like listicle)
- "Here's why..." throat-clearing before the actual content
- "The truth about..." (feels like clickbait without specificity)
- Two-sentence slides where the second sentence explains the first instead of advancing the idea
- Hook → wrong behavior → right behavior → test → moral close, unless the user explicitly asks for an arc
- Separate conclusion slides that only restate the lesson instead of delivering the final list item

## Cleanup Checklist

- [ ] Remove generic motivation.
- [ ] Remove empty throat-clearing.
- [ ] Remove unsupported claims.
- [ ] Remove copy that sounds like a template instead of the account.
- [ ] Remove every "remember that," "it's important to," "let's be honest."
- [ ] Verify each claim: "Would I say this to a real player I know?"
- [ ] Verify each slide has exactly one job.
- [ ] Verify the hook is specific enough that only a soccer player would stop for it.
- [ ] Verify no two slides say the same thing in different words.
- [ ] Verify the draft uses hook + list modules by default.
- [ ] Remove separate moral/conclusion slides unless the post explicitly needs one.
