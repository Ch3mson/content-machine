# Tools

Runnable utilities used by workflows.

| Tool | Path | Used by |
| --- | --- | --- |
| Image Sourcer | `image-sourcer/` | Workflow B when raw slide images are missing or a preset exists. |
| TikTok Photo Sourcer | `tiktok-photo-sourcer/` | Reference-account photo carousel downloads from TikTok. |
| Browser Harness | `browser-harness/` | Image Sourcer for Pinterest or browser-backed image search. |

Strategy docs, templates, research rules, and product files do not belong here.
