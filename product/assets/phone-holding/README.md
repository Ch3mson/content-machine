# Phone Holding — Product Shots

Sport-agnostic photos of the founder holding a phone with the w(inner) app visible.

## Purpose

These shots serve as the native product context layer across slideshows. They show
a real person holding a real phone — no mockups, no floating screens.

## Guidelines

- **Native grip** — phone held naturally, not staged like an ad.
- **Sport-agnostic** — no team colors, logos, or sport-specific gear visible.
- **App visible** — the screen should show the w(inner) app (any screen is fine).
- **Varied contexts** — different angles, lighting, and backgrounds so slides don't feel repetitive.
- **Real environment** — everyday settings (car, couch, hallway, field edge, etc.).

## Naming

Use the format: `{AR}-{sport}.jpg`

Examples:
- `9x16-basketball.jpg`
- `16x9-basketball.jpg`
- `9x16-running.jpg`
- `16x9-running.jpg`

When the same shot exists in both aspect ratios, the sport name stays the same and only the AR prefix changes.

## Usage

When adding a new shot, update the asset table in `product/app-brief.md` with the
filename and its approved slide role.
