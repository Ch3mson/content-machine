# w(inner) Product Assets

Place app-related images here.

Useful assets include:

- Athletes holding the app in a natural preparation context.
- App-in-hand images before training, before sleep, on a team bus, in a locker room, in a hotel, or near competition.
- Screenshots that show real functionality: athlete identity, diagnostic chat, radar chart, track summary, audio track, or reflection.
- Crops or mockups that can be used as subtle product context.
- Source images that help explain mental performance without making the slideshow feel sponsored.

Avoid:

- Generic phone mockups floating over backgrounds.
- App-store ad layouts.
- Download badges.
- Wellness or meditation imagery unless it is being used as an anti-reference.
- Product screenshots used as decoration without a slide-role reason.

When adding assets, update `product/app-brief.md` so each file has an approved use.
